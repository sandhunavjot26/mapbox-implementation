PRD IMPLEMENTATION RULES

1. Always follow PRD for UI behavior and naming.

2. Map must include:
   - Detection zones (larger radius)
   - Jamming zones (smaller radius)
   - Operational fences (polygon/line)
   - Drone tracks and paths

3. Alerts:
   - Critical → red
   - Medium → orange
   - Info → blue

4. Drone object must include:
   - ID
   - speed
   - altitude
   - direction
   - threat score

5. Support:
   - Minimal view
   - Detailed view

6. Path tracking:
   - Maintain history
   - Show trajectory line

7. Multi-drone:
   - Cluster visualization
   - Swarm indicators

8. UI must reflect real-time decision-making system
   (command center style)
